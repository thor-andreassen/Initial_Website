# Research Fellow Website

This is a responsive website showcasing the research of a postdoctoral research fellow.

## Features

- Responsive design that works on all devices
- Clean, professional layout suitable for academic profiles
- Smooth scrolling navigation
- Interactive elements with JavaScript
- Easy to customize content

## Files

- `index.html` - Main webpage structure
- `styles.css` - Styling for the website
- `script.js` - JavaScript for interactive elements
- `profile-placeholder.jpg` - Placeholder image for profile

## Customization

To customize this website:

1. Replace placeholder text with actual research fellow information
2. Update the profile image with a real photo
3. Modify colors in `styles.css` to match institutional branding
4. Add actual publications and research projects
5. Update contact information and social links

## Usage

Simply open `index.html` in any modern web browser to view the website.

## Structure

The website includes the following sections:
1. Header with name and title
2. Navigation menu
3. About section with profile information
4. Research interests section
5. Publications section
6. Contact information
7. Footer with copyright

## Responsive Design

The website uses a mobile-first approach with responsive breakpoints for:
- Mobile devices (small screens)
- Tablets (medium screens)
- Desktops (large screens)
